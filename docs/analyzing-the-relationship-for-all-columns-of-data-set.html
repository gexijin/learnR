st.matrix <- as.matrix(st)
ppc <- function(x, ...){ # A distance function of 1-PPC (Pearson's correlation coefficient)
as.dist(1 - cor(t(x), method = "pearson"))
}
heatmap(st.matrix, # data, must be a matrix!
scale = "column", # transformed to have zero mean and unit standard deviation by column
distfun = ppc, # use custom function to compute similarity between rows and columns
RowSideColors = rainbow(4)[sta$Region]) # add colored bar according to Species
legend("topleft", levels(sta$Region), fill = rainbow(4), cex = 0.45) # add legend in a separate line

The red color on the heatmap means higher value and yellow means the lower value for different obverved variables. Use only distance method(Pearson) for this heatmap.

Compare with other states, most northern, western states and several southern states which have higher Life.Exp, Income, HS.Grad and Frost have lower Area, Population, Murder and Illiteracy. On the contrary, lots of south states with lower Life.Exp, Income, HS.Grad and Frost have higher Area, Population, Murder and Illiteracy. Those states in the middle of the figure with black or dark red color have relatively higher value in all variables which we observed, like Arizona and California.

There are some special case like Alaska has real big area with relatively low life expetation, California has real big population with low frost, New York has high population, Hawaii and Florida have low frost, Louisiana has high illiteracy.

Income, HS.Grad and Life.Exp, together with Frost and Area build one cluster, while Illiteracy, Murder and Population build another cluster.
